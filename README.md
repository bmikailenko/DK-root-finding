Dk

Description: DK iteration root finding, input the polynomial coefficients and dk outputs the roots.

To make: 

	make

To run: 

	./dk [input file]

Example run: 

	./dk example.in

Example input file contents:

	0 3 4 7 1

Example input file contents means:

	y = 3x + 4x^2 + 7x^3 + x^4
